## 2. 插件安装与卸载
### (1)插件的概念
插件(Plug-in,又称addin、add-in、addon或add-on,又译外挂)是一种遵循一定规范的应用程序接口编写出来的程序。其只能运行在程序规定的系统平台下（可能同时支持多个平台），而不能脱离指定的平台单独运行。
### （2）插件的安装
1. 在VSCode界面找到左侧插件按钮，如图：
![](https://img-blog.csdnimg.cn/2019080710580154.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L2loZXl1,size_16,color_FFFFFF,t_70)
2. 可以通过搜索栏的搜索查找要安装插件(支持模糊查找,输入你要查找的关键词),如图：
![](https://img-blog.csdnimg.cn/20190807105823694.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L2loZXl1,size_16,color_FFFFFF,t_70)
3. 找到要安装的插件点击安装,如图：
![](https://img-blog.csdnimg.cn/20190807105857208.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L2loZXl1,size_16,color_FFFFFF,t_70)
4. 所有的插件都安装完了,如图:
![](https://img-blog.csdnimg.cn/20190807105924744.png?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L2loZXl1,size_16,color_FFFFFF,t_70)
### (3)插件的卸载
直接点击卸载按钮。
## 3. 在VSCode中使用Git管理代码
### （1）安装VSCode
### （2）安装git源代码管理系统
### （3）把VSCode和git相关联
通过VS Code的主菜单File -> Open File，打开本地库，这样VS Code就能使用Git进行源代码的管理了。

![](https://images2018.cnblogs.com/blog/628084/201807/628084-20180719165953484-1412384947.png)

打开“文件管理”，就能看到GITTEST本地文件夹，这是Git管理的本地库：

![](https://images2018.cnblogs.com/blog/628084/201807/628084-20180719171111366-1352768191.png)

打开“源代码控制”，可以看到SOURCE CONTROL是GIT，点击“√” 就可以提供代码的修改，而CHANGES列表中显式所有的代码修改：

![](https://images2018.cnblogs.com/blog/628084/201807/628084-20180719172239827-1773544650.png)

### (4)使用PowerShell脚本
使用PowerShell脚本来组装git命令，这样，只需要执行一段PowerShell脚本，就可以完成代码的提交和同步：
```git pull origin master
git add -A
git commit -m 'auto commit'
git push origin master
```
在VS Code中，执行PowerShell脚本，首先需要设置执行策略，以管理员权限启动PowerShell终端，输入以下命令，并输入A：

```Set-ExecutionPolicy RemoteSigned```

![](https://images2018.cnblogs.com/blog/628084/201807/628084-20180718165907861-1254386545.png)
